# OilCalcApp

Приложение для расчета потерь нефтепродуктов при транспортировке.

## Требования
- Xcode 14.0+
- iOS 16.0+

## Как запустить приложение

1. Откройте файл проекта `OilCalcApp.xcodeproj` в Xcode.
   ```bash
   open OilCalcApp.xcodeproj
   ```
2. В верхней панели Xcode выберите схему `OilCalcApp` и целевой симулятор (например, iPhone 15).
3. Нажмите кнопку **Run** (Play icon) или используйте комбинацию клавиш `Cmd + R`.

## Как запустить тесты

1. В Xcode выберите схему `OilCalcApp`.
2. Нажмите `Cmd + U` или выберите в меню `Product` -> `Test`.
3. Результаты тестов отобразятся в навигаторе тестов (Test Navigator, `Cmd + 6`).

## Структура проекта
- **Domain**: Бизнес-логика (калькуляторы, таблицы, валидация).
- **Presentation**: UI экраны (SwiftUI).
- **Infrastructure**: Работа с данными (история).
